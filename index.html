Steps to Install

Installing on Playdate Hardware

  1. Download the "8ball.pdx.zip" file.
  2. Unzip the downloaded file to extract the "8ball.pdx" file.
  3. Install "8ball.pdx" on your Playdate (check out the link below to learn how).
  4. Play and report bugs, suggest features, and share your experience to earn brownie points!
Learn more about sideloading at help.play.date

Playdate Console

Running in Playdate Simulator

  1. Download the "8ball.pdx.zip" file.
  2. Unzip the downloaded file to extract the "8ball.pdx" file.
  3. Make sure you have the Playdate SDK installed.
  4. Open the "8ball.pdx" file with Playdate Simulator.
  5. Play and report bugs, suggest features, and share your experience to earn brownie points!
Review the Playdate SDK documentation for more details
